Test Dashboard Access

Follow these steps to test the user dashboard:

Method 1: Direct Login Test

  1. Open browser console (F12)
  2. Run: localStorage.setItem('authToken', 'test-token')
  3. Run: localStorage.setItem('userRole', 'user')
  4. Refresh the page
  5. Look for "My Dashboard" in navigation
  6. Go to: http://localhost:4200/user-dashboard

Method 2: Login Form Test

  1. Go to: http://localhost:4200/login
  2. Enter username: testuser
  3. Enter password: password123
  4. Click Login
  5. Check navigation for "My Dashboard"

Method 3: Admin Test

  1. Go to: http://localhost:4200/login
  2. Enter username: admin
  3. Enter password: admin123
  4. Click Login
  5. Should see both "My Dashboard" and "Admin" links
